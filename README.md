# Dental Interactive Guide

The **Dental Interactive Guide** is a specialized, mobile-first social and clinical networking platform built for dental professionals, students, and patients. It combines the seamless, engaging user experience of a modern social feed with powerful, HIPAA-compliant clinical tools to foster education, discussion, and private consultations.

### ✨ Key Features
*   **Interactive Clinical Feed:** An edge-to-edge scrolling feed featuring multi-image carousels, double-tap to like, and threaded diagnostic discussions.
*   **Privacy-First Image Editor:** A built-in client-side editor allowing users to blur faces, redact text, and annotate X-rays *before* uploading.
*   **Diagnostic Polls:** Attach multiple-choice questions to clinical cases to gather consensus and gamify learning.
*   **Direct Messaging (DMs):** Secure 1-on-1 private messaging for specialist consultations, complete with image attachments and a "Share to DM" feature.
*   **Interactive Anatomy:** Clickable, user-generated hotspots on clinical images that map anatomical landmarks and pathologies.
*   **Progressive Web App (PWA):** Fully installable on iOS and Android devices for a native app experience.
